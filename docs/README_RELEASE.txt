АРКАНОИД - ИГРА ВЕРСИЯ 1.6.1
=====================================

Добро пожаловать в игру Арканоид!

ОСНОВНЫЕ ФУНКЦИИ:
- Классический геймплей Арканоид
- Система рекордов (топ-10)
- Звуковые эффекты и музыка
- Полная локализация на русский язык

УПРАВЛЕНИЕ:
←/→  - Движение платформы
M    - Отключение/включение музыки
ESC  - Выход из игры
H    - Просмотр рекордов
Enter - Подтверждение действия

СИСТЕМНЫЕ ТРЕБОВАНИЯ:
- Windows 7/10/11 (64-bit)
- Не требует установки дополнительного ПО

РАСПОЛОЖЕНИЕ ФАЙЛОВ:
Игра устанавливается в: %LOCALAPPDATA%\Games\Arkanoid
Рекорды сохраняются в: %LOCALAPPDATA%\Games\Arkanoid\highscores.json

ТЕХНИЧЕСКАЯ ИНФОРМАЦИЯ:
- Язык программирования: Python
- Библиотеки: Pygame 2.6+, NumPy
- Размер: ~30 МБ
