# Тесты проекта

Этот каталог содержит все тестовые файлы для проекта PyGame Ball.

## Структура тестов

### Основные тесты

- `test_interface_improvements.py` - Тестирование улучшений интерфейса игры
  - Логика отображения таблицы рекордов
  - Ограничения на 10 записей
  - Симуляция пользовательского интерфейса

- `test_interface_simple.py` - Упрощенный тест улучшений интерфейса
  - Логика работы с рекордами
  - Строгое ограничение на 10 записей
  - Изменения интерфейса

- `test_strict_10_limit.py` - Тест строгого ограничения на 10 записей
  - Проверка хранения максимум 10 записей в highscores.json
  - Работа с существующими рекордами

- `test_display_highscores.py` - Тест правильного форматирования таблицы рекордов
  - Проверка точного соответствия ширины столбцов заголовкам
  - Валидация разделителей и выравнивания данных
  - Автоматическое создание текстовых файлов для проверки форматирования
  - Тестирование пустой таблицы рекордов

- **Улучшена подсказка о продолжении**: "После ввода имени нажмите Enter для продолжения"

### Исправления тестов
- **Установлены зависимости**: pygame, numpy для корректной работы тестов
- **Исправлены пути импорта**: корректная работа во всех тестовых файлах
- **Исправлены логические ошибки**: правильная проверка ограничений на 10 записей
- **✅ Все 27 тестов проходят успешно**

### Модуль инициализации

- `__init__.py` - Делает директорию tests/ пакетом Python

## Запуск тестов

Для запуска всех тестов выполните:

```bash
# Запуск из корневой директории проекта
cd tests/
python test_display_highscores.py
python test_interface_improvements.py
python test_interface_simple.py
python test_strict_10_limit.py
```

## Требования

Для корректной работы тестов требуется наличие следующих модулей в корневой директории:
- `highscores.py` - Менеджер рекордов
- `PyGameBall.py` - Основная логика игры

Все тестовые файлы используют относительные пути для импорта модулей из корневой директории проекта.
