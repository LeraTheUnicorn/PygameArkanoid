# История изменений

## [1.6.0] - 2025-11-28

### Улучшения пользовательского интерфейса

- **Удалены дублирующие подсказки** - убраны подсказки "M - переключение звука" из экрана результатов игры и экрана
  таблицы рекордов
- **Улучшена подсказка о продолжении** - изменена с "Нажмите Enter для продолжения" на "После ввода имени нажмите Enter
  для продолжения"
- **Добавлена валидация ввода имени** - принимаются только буквы и цифры для имени игрока

### Технические улучшения

- Добавлена проверка допустимых символов
- Сохранена обратная совместимость с существующей функциональностью

### Тестирование

- Все существующие тесты проходят успешно (27 тестов)
- Код успешно компилируется без ошибок
- Проверена совместимость с системой рекордов

### Документация

- Обновлен README.md с описанием новых функций валидации
- Добавлена информация о визуальной обратной связи при вводе
- Обновлена секция управления с учетом улучшенных подсказок

## [1.5.1] - 2025-11-28

### Новые функции интерфейса

- Добавлен выход из игры по клавише ESC из любого места
- Добавлен выход из игры по крестику окна из любого места
- Добавлена возможность просмотра таблицы рекордов по клавише H на стартовом экране

### Архитектурные улучшения

- Обновлена сигнатура функции show_highscores() с добавлением параметра exit_on_esc
- Изменен тип возврата show_highscores() на tuple[bool, bool] (музыка, выход_из_игры)
- Добавлена умная навигация: ESC и Backspace ведут себя по-разному в разных контекстах
- Добавлена синхронизация состояния музыки между всеми экранами

### Тестирование

- Создана директория tests/ для централизованного хранения тестов
- Добавлен tests/README.md с документацией по тестам

### Документация

- Обновлено описание новых возможностей

### Исправления

- Исправлен запуск фоновой музыки (больше не запускается до экрана ввода имени)
- Убраны устаревшие звуковые файлы из каталога sounds
